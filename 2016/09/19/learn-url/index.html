学习URL

发表于 2016-09-19

组成

schema://host[:port#]/path/…/[?query-string][#anchor]

  • schema: 协议。包括 http、https、ftp 等。
  • host: HTTP 服务器的 IP 地址或者是域名。
  • port: 端口。可选。
  • path: 访问的资源的路径
  • query-string: 搜索串。是发送给服务器的数据。可选。
  • anchor: 锚。可选。
